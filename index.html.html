💡 使い方ステップ

  1. 手持ちの動画を読み込むか「デモ動画を生成」をクリック。
  2. 動画を停止させ、画面上の追跡したい物体（例：赤いボール）をクリック。色基準値が設定されます。
  3. 再生ボタンを押すと動画解析が始まり、物体の加速度データをグラフ化します。
  4. 解析を終えたらCSVデータ出力で数値を保存可能です。

🛠 精度を向上させるコツ

  • カメラをしっかり固定し、手ブレがない動画を使うこと。
  • 追跡対象と背景のコントラスト（色の違い）がはっきりしているもの。
  • 「物理スケール調整」には、動画内の実際の距離を入力すること（デフォルトは画面横幅を1.0mに仮定しています）。

📐 物理演算の定義

位置: P = (X, Y) [m]
速度: V = dP / dt [m/s]
加速度: A = dV / dt [m/s²]
重力加速度: G = |A| / 9.80665 [G]
